#  Global Currency Market Intelligence Dashboard

An end-to-end Data Analytics project that automates daily exchange-rate data collection using Python, stores historical data in MySQL, and delivers interactive business insights through an interactive Power BI dashboard.

##  Project Overview

This project automates the complete Exchange Rate Analytics workflow, from extracting live exchange-rate data to building an interactive dashboard for business users.

The solution integrates Python, MySQL, and Power BI to provide historical currency analysis, daily trend monitoring, and country-wise exchange-rate insights.


##  Business Problem

International businesses, import/export companies, traders, and finance teams rely on accurate exchange-rate information to support financial decisions.

Manually checking exchange rates from multiple websites is time-consuming and does not provide historical analysis.

This project solves that problem by creating an automated ETL pipeline that collects, stores, and visualizes exchange-rate data in one centralized dashboard.


##  Project Objectives

- Automate daily exchange-rate data collection.
- Maintain a historical exchange-rate database.
- Track daily currency movements.
- Build an interactive Power BI dashboard.
- Provide country-wise currency analysis.
- Create a scalable solution for future forecasting.
